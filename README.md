## Installation

```bash
# Using NPM
npm install @nodewell/assets --save-dev
```

## Usage

```bash
@nodewell/assets
  
  /logo
    /logo-only/...
    /name-only/...
      black.svg
      color.svg
      white.svg
    
    /logo-with-name/
      /horizontal/...
      /vertical/...
        black.svg
        color.svg
        white.svg
  
  /media
    /github
      avatar.png
      divider.svg
```
